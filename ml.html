Deep Learning

Battery SoC Prediction with LSTM Networks

Developed a Long Short-Term Memory (LSTM) recurrent neural network to predict the State of Charge (SoC) of lithium-ion battery cells in real time. The model was trained on dynamic discharge profiles combining current, voltage, and temperature signals.

Achieved <2% RMSE on held-out test data across multiple driving cycles, enabling more accurate BMS estimation for EV platforms at OLA Electric.

Bandgap Prediction

Gradient-boosted tree models trained on DFT datasets to predict bandgaps of chalcogenide perovskites from structural descriptors, reducing computational screening time by 10×.

Battery Degradation Modeling

Autoencoder-based models for detecting anomalous capacity fade in lithium-ion cells using electrochemical impedance signatures, enabling early-warning diagnostics in BMS.
